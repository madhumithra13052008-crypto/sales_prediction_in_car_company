 

# Car Sales Prediction Using Logistic Regression

## Project Overview

This project predicts whether a customer is likely to purchase a car based on factors such as Age and Salary.

The model is built using Logistic Regression and deployed using Streamlit.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Streamlit
* Pickle

## Machine Learning Algorithm

* Logistic Regression

## Features

* Customer purchase prediction
* Data preprocessing using StandardScaler
* Interactive Streamlit web application
* Model saving and loading using Pickle

## Project Workflow

1. Data Collection
2. Data Preprocessing
3. Feature Scaling
4. Model Training using Logistic Regression
5. Model Evaluation
6. Streamlit Web Application Development

## Model Performance

* Accuracy: 80%

## How to Run

1. Install the required packages

```bash
pip install pandas numpy scikit-learn streamlit
```

2. Run the Streamlit application

```bash
streamlit run sales project.py
```
or

```
python -m streamlit run "C:\Users\acer\OneDrive\Documents\project -1\sales project.py"
```
